# wakeUp-cloudtype

가난한 SJ는 cloudtype 프리티어를 사용하여 프로젝트를 하고 있다. 

그런데 cloudtype는 새벽만 되면 매일 자동으로 서버가 꺼져버린다..!

화가난 SJ는 새벽마다 서버를 다시 키는 또 다른 프리티어 서버를 가동시키는, 이른바 하청의 하청 프로젝트를 만들고야 만다.
